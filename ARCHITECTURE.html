Employee Cab App

An Expo (React Native) app for booking company cabs, with a real-time cab-tracking feature layered on Firebase. This is the system as it stands today — what's live, what's still demo data, and what's next.

Progress

✓

Stage 1 — Backend + Track screen

Firebase connected and verified; employee Track Cab screen shows a live location on a web map (free OpenStreetMap).

✓

Stage 2 — Cab moves

Driver simulator writes moving coordinates to Firebase; the cab drives toward the pickup in real time.

✓

Stage 3 — Real device GPS

Driver (live GPS) screen streams the phone's real location via expo-location. Proven live on web with a real position.

4

Stage 4 — Make it a product

Per-booking cab matching (each employee tracks their own assigned cab), ETA + route, the native Google map on phone via a dev build, and locking down the API key + database rules.
